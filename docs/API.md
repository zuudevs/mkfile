
# API

Dokumentasi ini menjelaskan bagian-bagian utama kode untuk pengembang.

- `src/main.cpp` - titik masuk aplikasi. Menangani parsing argumen singkat dan menjalankan flow utama.
- `lib/` - (jika ada) kumpulan library internal yang dapat digunakan ulang.

Tidak ada API publik berlapis (library) pada saat ini; proyek ini adalah aplikasi contoh kecil. Untuk kontribusi, tambahkan file header di `include/` atau library di `lib/` dan dokumentasikan fungsi/kelas baru di file ini.
